**Rossoforge-Service** A centralized and extensible time service for Unity that manages time scale, pause and resume, and provides access to the current date/time.

#
**Version:** Unity 6 or higher

**Dependencies:**
* [Rossoforge-Core](https://github.com/rossogames/Rossoforge-Core.git)
